# reference

External reference documentation for design inspiration.

## Purpose

This directory contains documentation from other Lisp implementations (primarily Janet) that informed Elle's design. These are NOT Elle's implementation — they are reference materials for understanding design decisions.

## Contents

Reference documents from Janet and other Lisps that influenced Elle's architecture, syntax, and semantics.

## Note for agents

When reading files in this directory, remember:
- These describe other languages, not Elle
- Elle may differ in implementation details
- For Elle's actual behavior, consult `../` (the main docs directory)
- For Elle's implementation, read the source code and AGENTS.md files in `src/`
